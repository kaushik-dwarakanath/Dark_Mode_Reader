This is a polished dark-mode Chrome extension that converts almost any website into a modern, high-contrast dark theme.

## Features

- Smart, global dark theme applied via CSS.
- Popup UI with a smooth toggle for the current tab.
- Per-user enablement state persisted with `chrome.storage.sync`.
- Minimal, performance-focused content script.

## How to Load in Chrome

1. Run Chrome and go to `chrome://extensions/`.
2. Turn on **Developer mode** (top-right).
3. Click **Load unpacked** and select this `Dark_Mode_Reader` folder.
4. Open any website and use the extension icon to toggle dark mode.
